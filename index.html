Қандай құрал ең көп көмектесті?

Осы жұмысты орындауда маған GitHub Models ең көп көмектесті. Оның HTML, CSS және JavaScript туралы түсіндірмелері өте анық және деталды болды. Мысалы, Flexbox пен CSS Grid айырмашылығын нақты мысалдармен түсіндіріп берді.

ЖИ нені дұрыс немесе ыңғайсыз жасады?

ЖИ HTML структурасын жасауда өте жақсы жұмыс жасады, бірақ кейде тым күрделі немесе контекске сай емес код ұсынды. Нақтырақ prompt жазу арқылы мұны түзетуге болады.

ЖИ 5 жылдан кейін әзірлеушілерді қалай өзгертеді?

Менің ойымша, 5 жылдан кейін ЖИ рутиналық код жазуды толық автоматтандырады. Әзірлеушілер синтакс жаттаудан гөрі архитектура жасауға, бизнес логикасын шешуге және ЖИ шығарған кодты тексеруге уақыт жұмсайды.
